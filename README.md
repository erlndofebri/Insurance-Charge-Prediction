# Insurance Charge Prediction
This project is trying to predict how much insurance premium of someone based on personal information. We tried to train our model using several models, and the result :

![Screen Shot 2022-08-23 at 01 12 05](https://user-images.githubusercontent.com/106853320/185990307-138af858-d133-4f01-9c3e-d5965481e6e7.png)

We trying to predict with new personal information data (manually created), with the information:
- age: 35 years old
- sex: female (encoded as 0)
- bmi: 26
- children: no children (encoded as 0)
- smoker: no smoking (encoded as 0)

and the result, insurance charge prediction of this person is : 5477.95
